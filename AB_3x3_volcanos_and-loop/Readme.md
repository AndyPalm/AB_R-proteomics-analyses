AB_3x3_volcanos_and-loop

Generates volcanos from 3x3 (3 control + 3 treatment replicates) per dataframe.

Uses "comparison results" from MSstats as input, and converts it to appear like Fragpipe replicate-level results.

Uses one unique .csv (dataframe contents) per comparison.

Loops through dataframes in a specified directory and generates volcano plots with consistent settings.

Built in R 4.2.2.
